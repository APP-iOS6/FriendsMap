# 🗺️ 프로젝트 소개
## 프로젝트 명
Friends map - 프맵

## ADS
내 사진과 친구들이 찍은 사진을 위치, 소개글, 시간과 함께 지도에서 볼 수 있는 앱

## 타겟층
스마트폰으로 사진을 찍어 공유하는 것을 좋아하는 10대 ~ 30대

## 페르소나
- 📱 인스타 스토리에서 보던 사진들이 어딘지 궁금했던 사람
- 👯 내 친구들이 현재 어디서 무엇을 하고 있는지 궁금한 사람
- 📆 서로간 위치의 근처에 있다면 당일에 약속을 잡고 친구를 만나고 싶은 사람

## 기능
- 간단한 소개글과 사진을 위치와 함께 업로드할 수 있습니다.
- 유저 친구들은 홈(지도)에서 유저의 사진을 볼 수 있으며 유저 또한 친구들의 사진을 볼 수 있습니다.
- 특정 친구의 사진 클릭시 사진을 크게 볼 수 있고 간단한 소개글, 위치, 시간도 알 수 있습니다.
